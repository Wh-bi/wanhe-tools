# Gumroad 数字产品创建清单

## 产品1：AI 工具箱 Pro 许可证（终身）
- **名称**：Wanhe AI Tools Pro - Lifetime License
- **价格**：$19
- **描述**：Unlock all Pro features across 10 extensions. One-time payment, lifetime access.
- **包含**：
  - AI Summarizer Pro (Deep 8b model + batch files)
  - AI Translator Pro (Full page + PDF translation)
  - Code Reviewer Pro (Batch review + custom rules)
  - Privacy Cleaner Pro (Scheduled auto-clean)
  - Page Monitor Pro (Unlimited monitors + email alerts)
  - Reader Mode Pro (Custom CSS + PDF export)
  - Smart Bookmarks Pro (Unlimited AI classify + sync)
  - Tab Manager Pro (Auto-sleep + rules + cloud backup)
  - Screenshot Tool Pro (Annotation tools + video recording)
- **交付方式**：License Key via email (Gumroad 自动发送)
- **封面图**：landing_page 截图

## 产品2：AI Summarizer Pro 许可证（单插件）
- **名称**：AI Summarizer Pro License
- **价格**：$3/月 或 $25/年
- **描述**：Deep 8b model, long summaries, batch file processing.

## 产品3：AI Translator Pro 许可证（单插件）
- **名称**：AI Translator Pro License
- **价格**：$1/月 或 $10/年
- **描述**：Full page translation, PDF translation, translation history.

## Gumroad 创建步骤（人类操作，30分钟）
1. 登录 https://app.gumroad.com
2. Products → New product
3. 填入上述信息
4. 上传封面图（用 landing_page 截图）
5. 发布
6. 在 Settings → Advanced → 开启 API 自动发货
