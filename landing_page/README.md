# Wanhe AI Tools

10 local AI browser extensions powered by Ollama + qwen3. Zero cloud costs, zero data leaks.

## Extensions

| # | Name | Free Features | Pro ($1-3/mo) |
|---|------|--------------|---------------|
| 1 | AI Summarizer | Fast mode, 3 levels | Deep 8b model, batch files |
| 2 | Privacy Cleaner | 4 data types, manual | Scheduled auto-clean |
| 3 | AI Translator | EN-ZH, ZH-EN, JA-ZH | Full page, PDF |
| 4 | Reader Mode | 3 themes, 4 fonts | Custom CSS, PDF export |
| 5 | Code Reviewer | 4 review dimensions | Batch, custom rules |
| 6 | Smart Bookmarks | AI classify 50, dedupe | Unlimited, sync |
| 7 | Page Monitor | 1 monitor, browser alert | Unlimited, email |
| 8 | Tab Manager | Merge, sleep, session | Auto-sleep, rules |
| 9 | Screenshot Tool | 3 modes, edit, download | Annotation, video |
| 10 | Password Generator | Crypto-grade, all types | - |

## Keywords

## Installation
```
1. Install Ollama: https://ollama.com
2. ollama pull qwen3:0.6b qwen3:8b nomic-embed-text
3. Start AI Toolbox backend (localhost:8888)
4. Load extensions in Edge/Chrome: edge://extensions > Developer mode > Load unpacked
```
