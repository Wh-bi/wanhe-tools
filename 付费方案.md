## 技术实现

### License Key 验证流程
```
用户购买 → Gumroad 发邮件(含Key) → 用户粘贴到插件 → 
插件 SHA256(Key + 设备ID) → 本地验证 → 解锁功能
```

### Key 生成
```
python -c "import uuid; print('SK-' + str(uuid.uuid4())[:12].upper())"
# 示例: SK-A3F8B2C1D5E9
```

### 防滥用策略
- 每个 Key 绑定最多 3 台设备
- 离线验证（本地SHA256，不依赖服务器）
- Key 黑名单内置在插件更新中
