# Edge Add-ons Submission - Password Generator

## Extension Name
Password Generator - Strong Random Passwords

## Short Description
Generate cryptographically strong passwords with one click. Adjustable length, char types, and strength meter. Zero permissions, fully local.

## Full Description
Password Generator creates truly random passwords using the Web Crypto API (crypto.getRandomValues). No pseudo-random math.random - military-grade randomness.

Features:
- Adjustable length: 8-64 characters
- Character types: uppercase, lowercase, numbers, symbols
- Real-time strength meter (weak/medium/strong) based on information entropy
- One-click copy to clipboard with confirmation
- Clean dark UI
- Zero external permissions - only clipboardWrite for copy
- No data collection, no storage, no tracking

## Search Keywords

## Category
Productivity

## Privacy Statement
This extension generates passwords entirely in the browser using the Web Crypto API. No passwords are stored, transmitted, or logged. The clipboardWrite permission is used only for the copy button.

## Permissions Justification
- clipboardWrite: Required to copy generated passwords to clipboard
