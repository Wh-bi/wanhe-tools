# Edge Add-ons Submission - Page Monitor

## Extension Name
Page Monitor - Webpage Change Tracker

## Short Description
Monitor any webpage for changes and get instant browser notifications. Track job boards, news, prices, and more.

## Full Description
Page Monitor automatically checks webpages you specify and notifies you when content changes. Perfect for tracking job postings, news updates, price drops, course availability, and any other web content you want to stay on top of.

Features:
- Add any URL to monitor
- Choose check interval: every 1 hour, 6 hours, or 24 hours
- Automatic text similarity comparison
- Browser notification when changes are detected
- Simple management dashboard
- Shows last check time and change status

## Search Keywords
page, monitor, track, change, notification, alert, watch, webpage, price, news, job

## Category
Productivity

## Privacy Statement
Page content is fetched and compared locally in the browser. No data is uploaded to any external server. Monitoring URLs are stored in browser local storage.

## Permissions Justification
- alarms: Schedule periodic page checks
- notifications: Alert user when changes are detected
- storage: Store monitoring tasks
- activeTab/scripting: Read current page for preview
