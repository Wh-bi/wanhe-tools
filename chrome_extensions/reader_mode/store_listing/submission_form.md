# Edge Add-ons Submission - Reader Mode

## Extension Name (English)
Reader Mode - Clean Reading View

## Extension Name (Chinese)
阅读模式

## Short Description
One-click strip clutter and read any webpage in a clean, distraction-free view. Font size and theme adjustable. Works offline.

## Full Description
Reader Mode transforms any cluttered webpage into a clean, distraction-free reading experience with a single click.

Simply click the extension icon and choose "Enable Reader Mode". The extension extracts the main content from the current page, removes ads, sidebars, navigation, and other distractions, and presents it in a beautifully formatted reading view.

Features:
- One-click content extraction
- Removes ads, sidebars, popups, and navigation clutter
- 3 reading themes: Dark, Sepia, Light
- Adjustable font size (16px-22px)
- Smart content detection (searches for article, main, and content containers)
- Works completely offline - no external dependencies

Perfect for: news articles, blog posts, technical documentation, research papers, and any content-heavy webpages.

## Privacy Statement
This extension does NOT collect, store, or transmit any user data. Content extraction happens entirely in the browser using the activeTab and scripting permissions. The reading view is opened in a local extension page. No external servers, no analytics, no tracking.

## Permissions Justification
- activeTab: Required to read and extract the current page content
- scripting: Required to inject content extraction script into the current page
