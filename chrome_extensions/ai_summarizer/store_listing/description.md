## Full Description
AI Web Summarizer lets you instantly summarize any webpage using your own locally running AI models. No cloud uploads, no API subscriptions, no data privacy concerns.

Features:
- One-click summarization of any webpage
- Two AI modes: Fast (qwen3:0.6b, ~6s) and Deep (qwen3:8b, higher quality)
- Three detail levels: Short / Medium / Long
- Fully local - runs on your Ollama server
- Clean dark-themed UI

Requirements:
- Ollama running locally with qwen3 models
- AI Toolbox backend (localhost:8888)
