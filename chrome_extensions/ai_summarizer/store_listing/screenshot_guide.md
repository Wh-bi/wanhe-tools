# Screenshot Guide

## Required Screenshots (for Chrome Web Store)

### Screenshot 1: Popup UI
- Open any webpage
- Click the AI Summarizer extension icon
- Screenshot the entire popup showing model/length selectors
- Resolution: 1280x800 or 640x400

### Screenshot 2: Summary on a news article
- Go to a news website (e.g., BBC News, Zhihu)
- Click Summarize
- Screenshot showing the summary result

### Screenshot 3: Summary on a technical page
- Go to a technical page (e.g., Wikipedia, GitHub README)
- Click Summarize
- Screenshot showing the summary result

## Tips
- Use Chrome DevTools device toolbar to set 1280x800 viewport
- Make sure the popup result area is visible
- Hide personal browser tabs/bookmarks
- Use a clean, uncluttered page as background
