# Edge Add-ons Submission - Tab Manager Pro

## Extension Name
Tab Manager Pro - Smart Tab Organizer

## Short Description
Search, merge, sleep, and save browser tabs. Reclaim memory and stay organized with one-click tab management.

## Full Description
Tab Manager Pro helps you take control of your browser tabs. Stop losing tabs in a sea of open windows.

Features:
- Search tabs by keyword across all windows
- Merge tabs from multiple windows into one with a single click
- Sleep inactive tabs to free memory (tabs stay visible, click to reload)
- Save current tabs as named sessions for later restoration
- Close individual tabs directly from the manager
- Real-time stats: total tabs, windows, sleeping tabs, estimated memory

Perfect for power users, researchers, and anyone who keeps 30+ tabs open.

## Search Keywords
tab, manager, organize, merge, sleep, memory, session, save, search, window

## Category
Productivity

## Privacy Statement
This extension operates entirely through browser APIs. No tab data, URLs, or page content is collected or transmitted. Session data is stored locally in browser storage.

## Permissions Justification
- tabs: Required to list, manage, and close browser tabs
- tabGroups: Required for future tab grouping features
- storage: Required to save tab sessions
- scripting: Required for tab-related operations
