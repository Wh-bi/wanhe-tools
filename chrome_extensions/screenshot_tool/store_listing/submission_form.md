# Edge Add-ons Submission - Screenshot Tool

## Extension Name
Screenshot Tool - Capture, Edit & Annotate

## Short Description
One-click screenshot with visible area, full page, and selection modes. Edit and download instantly.

## Full Description
Screenshot Tool is a powerful yet simple screenshot extension for your browser. No bloat, no cloud uploads - everything stays local.

Three capture modes:
1. Visible Area - Instantly capture what you see
2. Full Page - Capture the entire scrollable webpage
3. Select Area - Click and drag to capture exactly what you need

After capture, the editor opens with:
- Download as PNG
- Copy to clipboard (one-click)
- Image dimensions display

Keyboard shortcut: Ctrl+Shift+S

## Search Keywords
screenshot, capture, screen, image, clip, full page, visible, annotate, 截图

## Category
Productivity

## Privacy Statement
Screenshots are processed entirely in the browser. No images are uploaded to any server. Downloaded files go directly to your local machine.

## Permissions Justification
- activeTab/scripting: Required to capture tab content
- downloads: Required to save screenshots to disk
- clipboardWrite: Required to copy screenshots
