# 人类一次性操作（做完这些，后续全自动）

## 操作1：创建 GitHub 仓库（5分钟）
1. 打开 https://github.com/new
2. Repository name: wanhe-tools
3. Description: 10 Free & Open-Source AI Browser Extensions
4. Public OK
5. 不要勾选「Add a README file」（本地已有）
6. 点「Create repository」
7. 页面跳转后 → Settings → Developer settings → Personal access tokens → Tokens (classic)
8. Generate new token → 勾选 repo 和 workflow → 生成
9. 复制 Token → 发给事通：`Token: ghp_xxxxxxxx`

## 操作2：事通用 Token 自动完成（收到 Token 后）
- git push 全部代码到 GitHub
- 在仓库 Settings → Pages → Source: main branch / landing_page → Save
- 确认公网 URL: https://wanhe-tools.github.io/wanhe-tools/

## 操作3：创建 Gumroad 产品（30分钟）
打开 https://app.gumroad.com → 注册 → 逐个创建：

| 产品 | 价格 | 模板文件 |
|------|------|---------|
| AI Toolbox Pro (终身) | $19 | Gumroad产品页.md |
| Single Plugin License | $3/月 | 同上 |
| 5-Plugin Bundle | $9/月 | 同上 |

创建完后把购买链接发给事通。

## 操作4：安装 ngrok（可选，5分钟）
https://ngrok.com → 注册 → 下载 → 获取 Authtoken
发给事通 → 事通启动公网暴露 → 插件不需 Ollama 就能用
